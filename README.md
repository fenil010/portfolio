# Fenil Patel Portfolio

A responsive portfolio built with Next.js, React, TypeScript, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

The development server runs on `http://localhost:9002`.

## Checks

```bash
npm run lint
npm run build
```

## Project Structure

- `src/app` - App Router entry points and global Tailwind CSS.
- `src/components` - Navbar, hero, and portfolio sections.
- `src/lib/styles.ts` - Shared Tailwind class helpers.
- `public` - Images and SVG assets used by the portfolio.
